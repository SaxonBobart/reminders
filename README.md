# Reminders

An open-source, cross-platform clone of Apple Reminders.

## Vision

A 1:1 clone of Apple Reminders with a native UI on each platform — iOS, Android, Windows, and Linux. Free and open source under AGPL-3.0-or-later. An optional paid cloud sync service, backed by [Supabase](https://supabase.com/), will let users keep their reminders in sync across devices; running the app entirely locally is, and will remain, fully supported.

## Status

**Pre-alpha.** Nothing here works yet.

The current focus is an iOS-first local-only MVP. Other platforms and cloud sync will come later, once the core experience is solid on a single platform.
